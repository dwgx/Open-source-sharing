# Open-source-sharing

这个仓库是一个“历史压缩包整理站”。

简单说：
把以前保留的一些项目压缩包集中放在这里，方便统一留档和共享。

## 包含内容

仓库里当前主要是若干 `.zip` 压缩包，例如：

- `Autumn 1.8 src.zip`
- `EmperorMax.zip`
- `Kozi-java.zip`
- `LiquidWing NewSrc.zip`
- `src cat 1.0.9.9.zip`
- `tomknew src.zip`

## 使用方式

1. 下载你需要的压缩包
2. 解压后在本地查看源码
3. 按各项目自身说明进行构建

## 注意事项

- 这些内容主要用于学习与留档
- 不同压缩包的构建方式可能完全不同
- 请自行确认版权、许可和使用边界

## 声明

本仓库不提供违规用途支持，也不保证所有历史包都能直接运行。
